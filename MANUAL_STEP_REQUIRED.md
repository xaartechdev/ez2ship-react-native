## ⚠️ IMPORTANT: Manual Step Required

The EZ 2 SHIP logo PNG image that you attached needs to be manually saved as:
**`src/assets/images/ez2ship-logo.png`**

Please:
1. Save the attached PNG logo image to `src/assets/images/ez2ship-logo.png`
2. Delete the placeholder file `src/assets/images/ez2ship-logo.png.placeholder`
3. Run the icon generation script to create all app icons

### To generate app icons automatically:
```powershell
# On Windows (PowerShell)
.\generate-icons.ps1

# Or on Mac/Linux
chmod +x generate-icons.sh
./generate-icons.sh
```

### Manual icon generation (if you don't have ImageMagick):
Use online tools or design software to create the following icon sizes from your logo:

**iOS Icons:** 20x20, 29x29, 40x40, 58x58, 60x60, 76x76, 80x80, 87x87, 120x120, 152x152, 167x167, 180x180, 1024x1024
**Android Icons:** 48x48, 72x72, 96x96, 144x144, 192x192
